Jul 02 2019 -

Login with Email and Selection of Main Character

s of today, you need to log in with your registered email address and your password , both on the website and in the game. Your account name is no longer required.

Moreover, with today's server save, we have set the highest-levelled, unhidden character on your account as main character . This character will be used in the  friend list  after the release of the summer update to represent your account so that others can identify you.

For accounts that only had hidden characters, a new character has been automatically created as main character. For those that already had 20 characters, the hidden status of the lowest-levelled character has been removed to set it as main character.

If you want to change your main character you can do this for free once by using the Main Character Change voucher that has been assigned to your account. You can use this voucher in your account management page to choose a new main character. Please note that you cannot select characters that are hidden, scheduled for deletion or Tournament characters.

If you want to change your main character again, you can purchase a Main Character Change in the Store. Moreover, if you create a new character, you can also set it as main character for your account.
