Apr 14 2015 -
Fixes and Changes
ith today's server save, we implemented a few fixes and changes, for example:
  • An issue with a part of the eastern wall of the houses Wood Avenue 7 and 8 in Edron has been fixed.
  • From now on, the achievement "Natural Born Cowboy" can be obtained by riding any mount for the first time. This also includes mounts being bought via the Tibia payment system.
  • In Service Of Yalahar quest: We addressed a minor issue in the mission "Searoutes around Yalahar". It is not possible anymore to hand in the same item five times.
  • In addition, some graphical issues have been fixed.
Moreover, a whole new bunch of improved monster animations has been implemented.

Please note that a new client version is available. The patch will download automatically when you log into the game. You can also download the new stand-alone client version 10.77 from our download section .

See you in Tibia!
