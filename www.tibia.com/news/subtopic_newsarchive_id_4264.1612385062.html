BattlEye Information & Tibia 11 Fixes

ellow Tibians,
we have used the extended downtime to implement new features to our website.
From now on, you are able to see if a game world is protected by BattlEye. Therefore, we have added two new icons, which are displayed next to the corresponding game world name in your account management:

The game world is initially protected by BattlEye The game world protected by BattlEye since a certain date.

Those icons are also displayed in the game world overview and in the character creation interface. We have also added a "BattlEye Status" section in the world information, which will tell you since when the game world is protected by BattlEye.
Last but not least, it is now possible to filter by BattlEye protected game worlds in the character creation.

Additionally, we have added a new FAQ to our support interface.

Furthermore, we have released a few bugfixes for the Tibia 11 client. Among others, we changed the following:

  • Activating your numpad while walking now interrupts your character's movement.
  • It is now possible to set a hotkey for "Combat: Toggle Chase Mode" .
  • An issue causing the slider to reset while selling items to an NPC has been fixed.

Have a nice day in Tibia!
Your Community Managers
