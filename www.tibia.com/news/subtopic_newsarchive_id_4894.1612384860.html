Jan 09 2019 -

Bugfixes and Content Changes

ith today's server save, we have released a new client version. Among others, we changed the following:

  • The boss Grand Master Oberon is accessible again.
  • We have temporarily disabled the Frame Rate Limit slider for the Mac client.
  • The loot table of the crazed winter vanguard was slightly adjusted.
  • Tibians can no longer be turned into snowmen while standing in a protection zone.
  • Removed the protection zone block check from several portals and teleports that got implemented with the last winter update.
