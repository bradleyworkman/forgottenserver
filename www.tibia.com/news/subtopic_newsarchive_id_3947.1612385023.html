Tibia 11 Fixes

ith today's Tibia 11 client update, we changed the following:

  • An issue causing the client to crash is now fixed.
  • An issue, which -under certain circumstances- caused the character to stop while moving via map click and using hotkeys at the same time, is now fixed.
  • Turning your character should no longer result in additional steps in the opposite direction.
  • We have reset the keyboard delay to 250ms for every player. If you want to use your previous keyboard delay, you have to untick the "Use default delay (250ms)" box in your general options.
  • An issue, which caused the character to lose its target sometimes, is now fixed.

Have fun!
