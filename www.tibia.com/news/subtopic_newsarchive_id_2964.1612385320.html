Content Fixes and Changes
ith today's server save, we implemented a few content fixes and changes, for example:
  • The walking animation of characters has been adjusted once again.
  • We addressed an issue with the voting system in Rathleton. The dungeons should now work as intended again.
  • We fixed the problem with Rafzan's elevator and also added a ladder to offer another option to reach him.
  • The sandstone scorpion can be tamed again. Please note that the taming item still has a chance to break.
  • Also, some other graphic as well as map bugs and typos have been fixed.
Please note that a new client version is available. The patch will download automatically when you log into the game. You can also download the new stand-alone client version 10.60 from our download section .
