Mar 06 2018 -

Bugfixes and Content Changes

ith today's server save we have released a new client version. The following things have been fixed:

  • Players with an US phone number should now be able to receive a SMS with the verification TAN for our new recovery TAN system.
  • Scrolling through long Market histories, should no longer cause FPS drops.
  • Using the action bars should no longer cause FPS drops.
  • We removed several unobtainable items from the Cyclopedia.

Have fun,
Your Community Managers
