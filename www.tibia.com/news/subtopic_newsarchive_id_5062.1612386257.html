Two-Factor Email Code Authentication

s of today's server save, you can use Two-Factor Email Code Authentication as an alternative to using a Two-Factor Authenticator App. Both methods offer an additional layer of security to help prevent unauthorised access to your Tibia account. However, accounts can only be connected to one authentication method.

You can select one of the methods via your account management page. More information can be found in the manual .

Please also note that we will change the login process within the next month. To log in, you will then need your password and the account's email address . Your account name is no longer required. So make sure that your Tibia account is assigned to a valid email address!
