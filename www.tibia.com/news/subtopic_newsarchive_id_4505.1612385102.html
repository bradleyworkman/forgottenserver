Feb 27 2018 -

Redesigned Option Menu & Recovery TAN

rom now on, it is possible to link your mobile number to your Tibia account.
Tibians , who have registered a valid mobile phone number to their account, will receive a recovery TAN via SMS when they order a new Recovery Key. This recovery TAN can be used to change the email address of an account. Please note that the actual Recovery Key will still be sent via postal services.

With today's client update, we implemented a redesigned Options menu, which is now structured in main and corresponding sub categories. If you open the Options menu for the first time, only the most important settings are displayed. Click on " Show Advanced Options " at the bottom to get access to all available options.

The Hotkey menu is now split into the categories " General Hotkeys "," Action Bar Hotkeys " and " Custom Hotkeys " and offers a hotkey search function as well as a hotkey for " Combat: Toggle Secure Mode ". Please check out the updated Compendium entry in your client to learn more about the hotkey sections.

Last but not least, we added many missing items to the Cyclopedia.

We hope you like the new design,
