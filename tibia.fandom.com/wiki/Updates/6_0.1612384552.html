Summary

This update brought the first version of the current website, as well as many graphical changes.

See also: Updates Help

Changes

  • Tibia.com was improved with a forum .
  • Many old graphics were replaced.
  • It is no longer possible to have multiple clients running on the same computer.
  • The server has been moved to a new location, enabling better communication.
